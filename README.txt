Installation :
- Installer comme un addon normal (plus facile avec Minion)
- Aller dans le dossier d'addon (en général Documents\Elder Scrolls Online\live\AddOns\EsoRP)
- Lancer "Install EsoRP.exe"

Voilà ! Maintenant, pour lancer TESO avec ESO-RP, utilisez simplement le raccourci sur le bureau


Après celà, vous DEVEZ lancer le jeu via le nouveau raccourci sur le bureau permettant à l'addon de se synchroniser avec le serveur.
--------------------------

ESO-RP sera un addon permettant de faciliter le RP sur TESO en montrant les infos RP des autres joueurs (comme le fait Total RP 3).

Il founira des infos comme :
- la langue écrite (fr, en, de, ...)
- l'état actuel du personnage
- une description physique, morale, son histoire, etc ...
- et plein d'autre fonctions utiles !

Si vous être ici, soyez patient : une version officiel sortira sous peu
Imcompatible pour le moment avec Steam.
